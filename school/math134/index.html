Least Upper Bound Axiom

Any nonempty set \(S\) of real numbers that is bounded above so that there is \(x \in \mathbb{R}\) such that \(y \le x\) for all \(y \in S\) has a least upper bound, \(x := sup(S)\). That is, there is a unique smallest upper bound for S.

Proof: \(\sqrt{2}\) exists

  • Proof. Let set \(S = \{y \in \mathbb{R} : x^2 \le 2\}\).
    • \(S\) has an upper bound of 2. That is, \(\forall y \in S, x \le 2\). (Since if \(y \in S > 2\), then \(y^2 = 4 > 2\).)
    • \(S\) is non-empty. (For example, \(1 \in S\).)
  • Thus, according to the Least Upper Bound axiom, \(\exists x \in S\), which is the least upper bound of \(S\).
  • Now, we will show, by contradiction, that \(x^2 = 2\).

  • First, suppose \(x^2 < 2\). Let \(0 < \epsilon < 1\) be an arbitrary small number.
    • \[ \begin{align} (x + \epsilon)^2 & = x^2 + 2x\epsilon + a^2 \\ & < x^2 + 4\epsilon + \epsilon & (x \le 2 \Rightarrow 2x\epsilon < 4\epsilon \text{ and } 0 < \epsilon < 1 \Rightarrow \epsilon^2 < \epsilon) \\ & = x^2 + 5\epsilon \end{align} \]
    • Thus, we can choose an \(\epsilon\) such that \(x^2 + 5\epsilon < 2 \Rightarrow (x + \epsilon)^2 < 2\).
    • However, this contradicts the fact that \(x\) is the least upper bound of \(S\).

  • Second, suppose \(x^2 > 2\). Let \(0 < \epsilon < 1\) be an arbitrary small number.
    • \[ \begin{align} (x - \epsilon)^2 & = x^2 - 2x\epsilon + a^2 \\ & > x^2 - 2x\epsilon \\ & \ge x^2 - 4\epsilon & (x \le 2 \Rightarrow 2x\epsilon \le 4\epsilon \Rightarrow -2x\epsilon \ge -4\epsilon) \\ \end{align} \]
    • Since \(x^2 > 2\), we can choose \(\epsilon\) such that \(x^2 - 4\epsilon > 2 \Rightarrow (x - \epsilon)^2 > 2\).
    • However, this also implies that \(x - \epsilon > y, \forall y \in S\).
    • Thus, \(x - \epsilon\) is an upper bound for \(S\), contradicting the fact that \(x\) is its least upper bound.

  • From these contradictions, we can see that \(x^2 = 2\).

\(\epsilon\) – \(\delta\) Definition of a Limit

\(\displaystyle{\lim_{x \to a} f(x) = L}\) if for every \(\epsilon > 0\), there exists \(\delta > 0\) such that if \(0 < |x - a| < \delta\), then \(|f(x) - L| < \epsilon\).

Example limit proof: \(lim_{x \to 4} (2x - 5) = 3\)

Let \(\epsilon > 0\). We seek \(\delta > 0\) so that if \(0 < |x-4| < \delta\), then \(|2x - 5 - 3| < \epsilon\).

\[ \begin{align} |2x - 5 - 3| & = |2x - 8| \\ & = 2|x - 4| \\ & < \epsilon \\ & \Rightarrow |x - 4| < \frac{\epsilon}{2}\ \end{align} \]

Let \(\delta = \frac{\epsilon}{2}\). If \(|x - 4| < \delta\), consider

\[ \begin{align} |f(x) - L| & = |2x - 5 - 3| \\ & = 2|x - 4| \\ & < 2(\frac{\epsilon}{2}) \\ & = \epsilon \end{align} \]

Thus, \(0 < |x-4| < \delta\ \Rightarrow |2x - 5 - 3| < \epsilon), as desired.

Properties of Continuous Functions

Given a function \(f(x)\) continuous on \([a, b]\) (that is, continuous on every element of \([a, b]\))...

1. Intermediate Value Theorem

...and given \(c\) between \(f(a)\) and \(f(b)\), then there exists \(\exists y \in [a, b]\), with \(f(y) = c\).

In other words, \(f\) "takes on" all intermediate values between \(f(a)\) and \(f(b)\). The graph of \(f\) can be drawn without lifting your pencil from the paper; it cannot start below the x-axis and end above it without crossing it.
